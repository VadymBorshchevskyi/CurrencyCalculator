# CurrencyCalculator

This app has ability to convert amount in one currency to another, using exchange rate of a given date. Users can convert currency on the screen Convert and see history on the screen Histoty

Scenario 1: User enters the application, open the screen Convert, chooses the date, original currency, the amount in original currency and the target currency. The amount in the target currency will be shown. Every operation with converting saves in local db.
Scenario 2: User enters the application, opens the history screen and can see last 10 operations they performed.
